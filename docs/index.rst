Pillow
======

本文档是对Pillow官方文档的中文翻译。
本文档由phyng( `Github <https://github.com/phyng>` )翻译。
本文档翻译开始于2014-05-21.

Pillow是一个对PIL友好的fork，作者是 Alex Clark 和 Contributors. 而PIL是一个Python图像处理库，作者是 Fredrik Lundh 和 Contributors.
